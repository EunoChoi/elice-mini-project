## 컨벤션
- commit
  - `feat`: 새로운 기능 추가
  - `fix`: 버그 수정
  - `style`: UI/스타일 변경
  - `refactor`: 성능 및 기능 개선
  - `docs`: 문서 수정
  - `test`: 테스트 코드 추가/수정
  - `chore`: 기타 작업
- file, folder
  - component file : PascalCase
  - 일반 route file : kebab-case
  - 병렬 route file : camelCase
- variable
  - interface, type : PascalCase
  - 컴포넌트 function : PascalCase
  - 일반 function : camelCase
